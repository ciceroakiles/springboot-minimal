# springboot-minimal

"Pequeno" projeto com algumas camadas prontas.  
Feito apenas com o propósito de se tornar escalável.

Dependências (Maven):
* Lombok;
* H2 Database;
* Spring Web;
* Spring Boot DevTools;
* Spring Data JPA.

Na IDE? _Live reloading_ OK  
Fora da IDE? ```mvnw spring-boot:run``` (Windows)
